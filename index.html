Executive Summary

This document presents a structured internship onboarding and management system for Nexbyt Technology designed to improve recruitment, onboarding, communication, and performance tracking.

Task 1: Internship Process Review

Application Process

Internship applications are received through designated communication channels, Applicants submit their personal details, academic background, and area of interest.

Applications are then forwarded to the HR team for initial review.

Candidate Evaluation

Candidates are assessed based on:

  • Academic qualifications
  • Technical or relevant skills
  • Communication ability
  • Suitability for available departments

Shortlisted candidates are selected for the next stage.

Acceptance Process

Successful candidates are notified via email. The acceptance message includes:

  • Internship confirmation
  • Assigned department
  • Supervisor details
  • Start and end dates

Onboarding Process

New interns are onboarded through:

  • Internship Welcome Guide
  • Intern Handbook
  • Introduction to supervisor
  • Communication channel setup

Interns are then assigned initial tasks.

Performance Tracking

Intern performance is monitored through:

  • Task Completion
  • Attendance
  • Communication with supervisors
  • Quality of output
  • Weekly feedback reports

CHALLENGES IDENTIFIED

Lack of Centralized System

Intern data is not stored in a unified database, making tracking difficult.

Scalability Issues

The current process may not efficiently support a large number of interns, especially internationally.

Inconsistent Evaluation

Different supervisors may use different standards when assessing intern.

RECOMMENDATIONS

Centralized Intern Databas

A structured database (Excel) should be implemented to store and manage all intern records.

Standard Evaluation Framework

A unified scoring system should be introduced to ensure fairness and consistency in performance evaluation.

Performance Tracking System

A simple HR dashboard or tracking sheet should be introduced to monitor:

  • Attendance
  • Task Completion
  • Weekly performance
  • Supervisor feedback

CONCLUSION

The current internship process at Nexbyt Technology provides a good foundation for learning and development. However, improvements are required in automation, standardization, and centralized data management.

Implementing the recommended systems will improve efficiency, ensure consistency in evaluation, and allow the internship program to scale effectively to support both local and international intern.
